# 🚀 Pointify Dashboard

Pointify is a modern, high-performance SaaS dashboard designed to automate and streamline your daily search rewards quest. Built with efficiency and aesthetics in mind, it provides a premium interface for managing daily search tasks and tracking rewards progress.

### 🔗 [View Live Demo](https://vishwajeetchandole.github.io/pointify-auto-search)

![View Live Demo](https://raw.githubusercontent.com/vishwajeetchandole/pointify-auto-search/main/preview.png)

## ✨ Key Features

- **⚡ Auto Search Engine**: Complete your entire daily quest with a single click. Our paced automation system cycles through tasks in a dedicated tab to ensure maximum reliability and compatibility.
- **🎨 Premium SaaS UI**: A state-of-the-art dashboard built with **Tailwind CSS v4**, featuring glassmorphism, smooth micro-animations, and a sleek dark/light aesthetic.
- **📊 Real-time Stats**: Track your daily searches, current points, and goal progress with visual indicators.
- **🔄 Dynamic Task Generator**: Instantly refresh your quest list with a wide range of trending topics in AI, technology, and global markets.
- **✨ Visual Feedback**: Interactive search items with success badges and emerald-themed completion states.
- **💎 Modern Typography**: Perfectly balanced interface using **Space Grotesk** for display and **DM Sans** for body text.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Bundler**: [Vite 7](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks (useState, useEffect)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vishwajeetchandole/pointify-auto-search.git
   ```

2. Navigate to the project directory:
   ```bash
   cd pointify-auto-search
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

### 📖 Usage

1. Open the dashboard in your browser (default: `http://localhost:5173`).
2. **Manual Search**: Click the "Search" button on any task to open it in a dedicated tab.
3. **Auto Search**: Click the violet **"Auto Search"** button to automatically cycle through all tasks in the list.
4. **Refresh**: Use the "Refresh Tasks" button to generate a new set of quest items.
